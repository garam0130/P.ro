Timeline

피로그래밍의 피로했던 발자취.

  • NOV 2014

    LET'S MAKE IT

    첫째 K와 셋째 K는 비전공자를 위한 코딩 동아리를 만드는데 의기투합했다. 둘째 K가 여기에 합류했다. 기존의 프로그래밍 동아리는 전공자 수준의 지식을 요구했기 때문에 문턱이 높았다. 또한 막 생겨나던 비전공자를 위한 동아리들은 높아지는 프로그래밍 교육 열기를 감당하기 어려웠다. 우리도 코딩을 배우면서 동아리를 제대로 만들어보자는 생각에 본격적인 준비에 돌입하였다.

  • DEC 2014

    P.rogramming 1st

    Python + Flask 웹 프레임워크.
    3개 대학에서 모인 18명의 학생들.
    피로그래밍 1기는 그렇게 시작했다.

  • FEB 2015

    Be The Unicorn

    첫 해커톤이 열렸다.
    코드네임 Be The Unicorn.
    그리고 6개의 웹 서비스가 만들어졌다.

  • MARCH 2015

    P.rogramming 2nd

    Java + Android 프로젝트.
    3개 대학에서 모인 30명의 학생들.
    피로그래밍의 두번째 기수가 출범했다.

  • JULY 2015

    P.rogramming 3rd & M.T

    Python + Django 웹 프레임워크.
    여름방학이 시작하면서 3기를 모았다.
    4개 대학에서 27명이 모였다.
